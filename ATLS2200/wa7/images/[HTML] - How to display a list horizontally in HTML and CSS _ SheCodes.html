[HTML] - How to display a list horizontally in HTML and CSS

Learn how to present a list horizontally in HTML and CSS by using the display: inline-block property on list items.

👩‍💻 Technical question

Asked 7 months ago in HTML by Esme 
how do i present a list horizontally instead of vertically?

🤖 SheCodes Athena says:

Answered in 2.32 seconds

To present a list horizontally instead of vertically in HTML and CSS, you can use the display: inline-block property on the list items.

Example in CSS:

css
 
 
Copy code


ul {
  list-style: none;
}

li {
  display: inline-block;
  margin-right: 10px; /* add spacing between items */
}

Example in HTML:

html
 
 
Copy code


<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

This will display the bulleted list items horizontally with some spacing in between them. You can adjust the spacing by changing the value of margin-right.

what are all html elements?

Answered in 3.27 seconds

HTML stands for Hypertext Markup Language, and it is the standard markup language used for creating web pages. HTML elements are the building blocks of an HTML document. They define the structure and content of a webpage.

There are many HTML elements, but here are some commonly used ones:

  • <html>: Defines the root element of an HTML page.
  • <head>: Contains metadata about the HTML document.
  • <title>: Sets the title of the HTML document, which appears in the browser's title bar.
  • <body>: Contains the visible content of the HTML document.
  • <h1> to <h6>: Defines headings of different sizes.
  • <p>: Defines a paragraph.
  • <a>: Creates a hyperlink to another webpage or a specific location on the same webpage.
  • <img>: Inserts an image into the webpage.
  • <ul>: Creates an unordered list.
  • <ol>: Creates an ordered list.
  • <li>: Defines a list item.
  • <div>: Creates a division or a container for other HTML elements.
  • <span>: Defines a section within a document.
  • <table>: Creates a table.
  • <form>: Creates an HTML form for user input.

This is just a small selection of HTML elements. You can find a comprehensive list of HTML elements and their descriptions on the Mozilla Developer Network (MDN) documentation: HTML elements

php
 
 
Copy code

if($_SERVER['REQUEST_METHOD'] == 'POST') {

In HTML, <span> and <div> are both container elements, but they serve different purposes.

  • <span> is an inline element that is typically used to group and style a specific part of text within a larger block of content. It is often used for applying CSS styles or for scripting purposes. html
 
 
Copy code


<div class="container">

  <p>This is the main content of the page.</p>
</div>

In summary, <span> is used for inline styling or scripting within a text, while <div> is used for creating sections or divisions within a webpage.
